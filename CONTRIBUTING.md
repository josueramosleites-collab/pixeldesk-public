# Contributing to PixelDesk

Thanks for being here. This repo holds the **public docs and agent-connection guides** for PixelDesk. The main app code is in a separate private monorepo for now (will open progressively).

## What you can do here

| Action | Where |
|---|---|
| 💬 Ask a question, share an idea | [Discussions](https://github.com/josueramosleites-collab/pixeldesk-public/discussions) |
| 🐛 Report a bug in the docs | [Issues](https://github.com/josueramosleites-collab/pixeldesk-public/issues) |
| 📝 Improve a doc, fix a typo, add an example | Open a Pull Request |
| 🎨 Suggest a new visual theme | Discussions → Ideas |
| 🔌 Share a new integration recipe | PR adding to `CONNECT-AGENT.md` |

## Pull Request flow

1. Fork the repo
2. Create a branch: `git checkout -b docs/your-thing`
3. Make changes locally — markdown only here, no code-build needed
4. Commit with clear message: `docs: explain how to connect Aider`
5. Push to your fork and open a PR pointing at `main`
6. We review within a few days

## What we'll merge fast

- Typo fixes
- New code examples for connecting agents (Python, Go, Rust, Ruby, etc.)
- Better explanations
- New integration recipes
- Translations (if you can do pt-BR/es/fr/zh/ja/ko/hi)

## What we'll discuss before merging

- New sections in README
- Changes to how endpoints are described
- Anything touching pricing/licensing

## What stays out of scope here

This is the **public docs repo**. Not the main app. So we don't merge:

- App source code (lives in private repo)
- Feature requests for the app (open Discussion instead — we'll see it)
- Bug reports about pixeldesk.ai itself — use [pixeldesk.ai/contato](https://pixeldesk.ai/contato)

## Code of conduct

Be kind. We're building this in public because we want a community, not a fan club. Disagreement is welcome; rudeness is not.

If something feels off in a discussion, ping a maintainer (`@josueramosleites-collab`) and we'll handle it.
