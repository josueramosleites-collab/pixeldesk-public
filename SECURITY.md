# Security Policy

## Reporting a vulnerability

Found a security issue on **pixeldesk.ai** or in the docs of this public repo?

**Please do NOT open a public issue.**

Email instead: [via pixeldesk.ai/contato](https://pixeldesk.ai/contato) (mention "security" in subject)

We'll:
- Acknowledge receipt within 48h
- Investigate and respond with a timeline
- Credit you (with your permission) in the fix announcement

## In-scope

- pixeldesk.ai production app
- API endpoints documented in this repo
- Examples in CONNECT-AGENT.md that could leak credentials if followed

## Out-of-scope

- Third-party services (Anthropic, Vercel, etc.)
- Social engineering
- Issues that require physical access to the user's machine

Thanks for helping keep the community safe.
